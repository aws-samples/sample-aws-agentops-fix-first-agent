    "## How to Perform A/B Testing\n",
    "\n",
    "A/B testing in AgentCore splits live production traffic between two variants and continuously evaluates performance with statistical significance. The **AgentCore Gateway** handles traffic routing — your agent code does not change.\n",
    "\n",
    "### How it works\n",
    "\n",
    "1. **Create an A/B test** — specify a Gateway, two variants (control & treatment), traffic weights, an online evaluation config, and an IAM execution role.\n",
    "2. **Start the test** — the Gateway splits incoming traffic based on the runtime session ID. Assignment is sticky (same session → same variant).\n",
    "3. **Online evaluation scores each session** — evaluators run against each completed session and map scores to variants.\n",
    "4. **Statistical significance is computed** — per-evaluator metrics: mean score, p-value, confidence interval, significance flag. p < 0.05 = significant.\n",
    "5. **Stop & deploy the winner** — route 100% traffic to the winning variant.\n",
    "\n",
    "### Two Variant Patterns\n",
    "\n",
    "AgentCore supports two A/B test patterns:\n",
    "\n",
    "#### Pattern 1: Target-Based Variants\n",
    "\n",
    "Use when the change involves **code changes, framework upgrades, or entirely different agent implementations**.\n",
    "\n",
    "Each variant routes to a **different AgentCore Gateway target** pointing to a different runtime endpoint.\n",
    "\n",
    "- Variant configuration: `variantConfiguration.target` with the Gateway target name\n",
    "- Evaluation: `perVariantOnlineEvaluationConfig` (one online eval config per variant — each endpoint has its own log group)\n",
    "- Gateway filter: `gatewayFilter.targetPaths` scopes which paths the A/B test intercepts\n",
    "\n",
    "**Example use cases:** \n",
    "- Testing a Strands agent vs a LangGraph agent\n",
    "- Validating a major code refactor\n",
    "\n",
    "#### Pattern 2: Configuration Bundle Variants\n",
    "\n",
    "Use when the change is **purely configuration** (system prompt, model ID, tool descriptions).\n",
    "\n",
    "Both variants run on the **same AgentCore Runtime** with different configuration bundle versions. The Gateway injects the bundle reference via W3C baggage headers.\n",
    "\n",
    "- Variant configuration: `variantConfiguration.configurationBundle` with the bundle version\n",
    "- Evaluation: `sharedOnlineEvaluationConfig` (single shared config — both variants share the same log group)\n",
    "- No separate endpoints needed\n",
    "\n",
    "**Example use cases:**\n",
    "- Testing a new system prompt\n",
    "- Comparing model parameters (temperature, top-p)\n",
    "\n",
    "#### Choosing a Pattern\n",
    "\n",
    "| Aspect | Target-based | Configuration bundle |\n",
    "|--------|-------------|---------------------|\n",
    "| What varies | Entire runtime hosting (code, framework, model) | System prompt, tool descriptions, model parameters |\n",
    "| Routing | Different targets per variant | Same target, different config bundles |\n",
    "| Evaluation config | One online eval config per runtime | Single shared online eval config |\n",
    "| When to use | Code changes, framework upgrades | Configuration-only changes |"
   ]

    "import os, shutil, subprocess, json, time, uuid\n",
    "\n",
    "# --- Helper functions ---\n",
    "def run_cmd(cmd):\n",
    "    \"\"\"Run a shell command, return stdout. Raise on failure.\"\"\"\n",
    "    r = subprocess.run(cmd, capture_output=True, text=True, shell=True)\n",
    "    if r.returncode != 0:\n",
    "        raise RuntimeError(f'Command failed: {cmd}\\n{r.stderr}')\n",
    "    return r.stdout.strip()\n",
    "\n",
    "def get_param(name):\n",
    "    \"\"\"Read an SSM parameter by short name.\"\"\"\n",
    "    return run_cmd(f'aws ssm get-parameter --name /{APP_NAME}/{name} --query Parameter.Value --output text')\n",
    "\n",
    "def package_agent(agent_dir):\n",
    "    \"\"\"Package agent source + deps into build/ for AgentCore Runtime.\"\"\"\n",
    "    build_dir = os.path.join(agent_dir, 'build')\n",
    "    if os.path.exists(build_dir):\n",
    "        shutil.rmtree(build_dir)\n",
    "    os.makedirs(build_dir)\n",
    "    subprocess.run([\n",
    "        'uv', 'pip', 'install',\n",
    "        '--python-platform', 'aarch64-manylinux2014',\n",
    "        '--python-version', '3.12',\n",
    "        '--target', build_dir,/\n",
    "        '-r', os.path.join(agent_dir, 'requirements.txt')\n",
    "    ], check=True)\n",
    "    # Copy agent source code (includes bin/opentelemetry-instrument)\n",
    "    src_dir = os.path.join(agent_dir, 'src')\n",
    "    for item in os.listdir(src_dir):\n",
    "        s = os.path.join(src_dir, item)\n",
    "        d = os.path.join(build_dir, item)\n",
    "        if os.path.isdir(s):\n",
    "            shutil.copytree(s, d, dirs_exist_ok=True)\n",
    "        else:\n",
    "            shutil.copy2(s, d)\n",
    "    # Remove Windows .exe wrappers that uv generates in bin/\n",
    "    bin_dir = os.path.join(build_dir, 'bin')\n",
    "    if os.path.exists(bin_dir):\n",
    "        for f in os.listdir(bin_dir):\n",
    "            if f.endswith('.exe'):\n",
    "                os.remove(os.path.join(bin_dir, f))\n",
    "    return build_dir\n",
